# Changelog — 1 мая 2026

## Pomodoro FAB + Геймификация

### Новый интерфейс таймера

Постоянно раскрытый блок Pomodoro справа на dashboard удалён — kanban получил всю ширину экрана.

Вместо него появился **Floating Action Button** в правом нижнем углу с минималистичной иконкой-помидоркой. Клик открывает панель поверх контента, не скрывая задачи и колонки.

Когда таймер активен, в кнопке появляется счётчик времени и пульсирующий красный индикатор. На паузе — счётчик замирает.

### Рабочий Pomodoro-таймер

- **Start** — запуск из карточки задачи.
- **Pause / Resume** — приостановить и продолжить без потери прогресса.
- **Stop** — завершить сессию, время записывается в задачу.
- **Восстановление после обновления страницы** — активная сессия хранится на сервере, таймер продолжается там, где остановился.

### Панель «Сегодня»

В попапе — три метрики за текущий день:
- Закрыто задач
- Время за работой
- Серия активных дней (streak)

**Колесо прогресса** показывает сколько помодоро сделано из дневной цели (по умолчанию 8).

### Геймификация — Достижения

7 стартовых достижений с прогресс-баром и эффектом unlock:

| Достижение | Условие |
|---|---|
| 🍅 First Tomato | 1 завершённая pomodoro-сессия |
| 🔥 On a Roll | 10 pomodoro-сессий |
| 🏆 Pomodoro Master | 100 pomodoro-сессий |
| ⚡ 3-Day Streak | 3 дня активности подряд |
| 🌟 Week Warrior | 7 дней активности подряд |
| ✅ Task Crusher | 25 закрытых задач |
| 🎯 Daily Goal! | Выполнить дневную цель pomodoro |

При разблокировке — тост-уведомление с иконкой и описанием.

### Изменения на бэкенде

- `GET /api/v1/time-entries/active` — текущая активная сессия.
- `POST /api/v1/time-entries/pause` / `/resume` — пауза/продолжение с корректным накоплением времени.
- `GET /api/v1/achievements` — список достижений с прогрессом.
- `GET /api/v1/analytics/overview` — расширен: добавлены `current_streak_days` и `daily_pomodoro_goal`.

### Изменения в БД

- `time_entries`: новые поля `paused_at` и `accumulated_seconds`.
- Новые таблицы `achievements` (справочник) и `user_achievements` (прогресс юзеров).
- `users`: новое поле `daily_pomodoro_goal INT DEFAULT 8`.

### Мелкие исправления

- Исправлен сломанный URL старта таймера (`/time-entries/start` → `/time-entries`).
- Починен snake_case/camelCase mismatch в `analyticsApi.getOverview`.
