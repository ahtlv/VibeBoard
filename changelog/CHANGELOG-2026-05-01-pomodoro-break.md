# Changelog — 1 мая 2026

## Pomodoro — авто-перерыв, локализация, UX

### Авто-перерыв после рабочей сессии

После завершения 25-минутной сессии автоматически запускается 5-минутный перерыв — без каких-либо действий со стороны пользователя. По окончании перерыва — гонг и возврат в idle.

Реализована отдельная фаза `break` в state machine таймера. Фаза влияет на цвета, анимации, подписи кнопок и поведение API-вызовов (перерыв не пишется в `time_entries`).

### Визуальное разделение работы и отдыха

- Над таймером появился лейбл фазы: `🍅 Работа` / `☕ Перерыв`
- Цифры во время перерыва — зелёные (`emerald`)
- Кнопка «Стоп» в режиме перерыва заменена на `⏭ Пропустить` (зелёного цвета)
- Кнопка «Продолжить» и pulse-glow на FAB тоже зелёные в режиме перерыва
- В строке заголовка браузера — `☕ 04:55 — VibeBoard` во время перерыва

### Спящая помидорка

В режиме перерыва FAB-помидорка «засыпает»:
- Медленное дыхание (`scale 0.95↔1.05`, 3 с)
- Три буквы Z разного размера поднимаются вверх и фейдятся с задержками 0 / 0.8 / 1.6 с — непрерывный цикл
- В рабочем режиме — прежняя idle-анимация покачивания

### Гонг при завершении сессии

При истечении 25-минутного таймера звучит гонг (`/sounds/gong.mp3`). Тот же звук играет по окончании перерыва.

### Моментальное открытие popover

Данные (ачивки и today-stats) теперь загружаются в фоне при старте приложения (`PomodoroProvider`), а не при клике на FAB. При открытии поповера все элементы видны сразу, без поэтапного «вспрыгивания».

Дополнительно: если данные ещё не пришли к моменту клика (первые ~300 мс после загрузки) — поповер ждёт ответа API и только потом раскрывается. Максимальное ожидание — 400 мс, после чего открывается с skeleton-плейсхолдерами нужного размера.

### Плавная анимация раскрытия поповера

Поповер теперь «вырастает» из FAB-кнопки: `scale 0.95 + opacity 0 → scale 1 + opacity 1` за 200 мс. `transform-origin: bottom right`.

### Оптимизация бэкенда

В `achievementsService.list()` функция `computeStreak()` вызывалась последовательно после первого батча из 4 запросов, хотя не зависела от их результатов. Перенесена в общий `Promise.all` — лишний RTT к Supabase устранён, эндпоинт `/achievements` отвечает примерно вдвое быстрее.

### Локализация Pomodoro-поповера

- Кнопки: `▶ Старт`, `⏸ Пауза`, `⏹ Стоп`, `▶ Продолжить`, `⏭ Пропустить`
- Подпись кольца: `цель дня`
- Формат времени в today-stats: `0ч 9м` (вместо `0h 9m`)
- Тосты и браузерные уведомления при завершении сессии — на русском
- Названия и описания всех ачивок переведены в БД
