# Changelog — 1 мая 2026

## Activity Heatmap — тепловая карта активности

### Что появилось

- На странице Календарь добавлена тепловая карта активности за последний год — квадратики по дням, цвет зависит от количества выполненных задач
- Сегодняшний день отображается слева (новые дни — слева, прошлое — справа)
- Данные берутся из поля `completed_at` задач со статусом `done`
- Показывается общее число задач за год и рекорд за один день

### UX и визуал

- Неделя начинается с понедельника
- Отображаются метки дней: пн, ср, пт (локализованные)
- Названия месяцев локализованы (ru/en)
- Тултип при наведении показывает число задач и дату в локальном формате: «5 задач · 1 мая 2026 г.»
- Правильные склонения в тултипе: задача / задачи / задач
- Легенда интенсивности (Меньше → Больше) в шапке

### Backend

- Новый эндпоинт `GET /api/v1/tasks/heatmap?workspace_id=X`
- Участники (member) видят только свои задачи (созданные или назначенные), владельцы и администраторы — все задачи workspace
- Группировка по UTC-дате `completed_at`, диапазон — последние 365 дней

### Технические правки

- Исправлено выравнивание лейблов месяцев (ширина колонки = 14px = w-3 + gap-0.5)
- Ячейки увеличены до `w-3 h-3` (12px), равномерный gap 2px по обоим осям
- Короткие лейблы последнего неполного месяца скрываются (span < 2)
