# Changelog — 30 апреля 2026

## Создание и редактирование задач через попап

### UX

- Кнопка `+ Add task` внизу колонки убрана. Вместо неё — круглая иконка **+** в заголовке колонки, правее счётчика задач. Единое место для добавления на всех досках.
- Клик на `+` открывает универсальный попап (`TaskModal`) в режиме **create**.
- Клик на карточку задачи открывает тот же попап в режиме **edit**.

### Форма создания / редактирования

Поля: title, description, priority (low / medium / high / urgent), due date, **цвет фона карточки**, **assignees**.

- Цвет фона — палитра из 8 приглушённых тонов (rose, amber, lime, sky, violet, pink, teal, slate). Выбранный цвет применяется к фону самой карточки и сохраняется в базе.
- Assignees — список участников доски с мульти-выбором (кликабельные плашки с аватаром / инициалами). Аватары назначенных пользователей отображаются на карточке (стек до 3 + overflow).
- Pomodoro-таймер, checklists, recurring — остаются в edit-режиме как прежде.

### Drag & Drop

Перетаскивание задач теперь поддерживает **точную позицию** внутри колонки: между карточками появляются drop-зоны, которые подсвечиваются при наведении. Задача встаёт именно туда, куда её бросили.

### Backend

- Новое поле `bg_color` в таблице `tasks` — хранит ID цветовой темы.
- Новая таблица `task_assignees (task_id, user_id, board_id)` — связь задач с участниками доски.
- `POST /tasks` и `PATCH /tasks/:id` — принимают `bg_color`, `assignee_ids`, полный набор полей (ранее `POST` принимал только title).
- `GET /boards/:id` — загружает assignees одним доп. запросом и возвращает `assignee_ids` в каждой задаче.
- Priority enum расширен: добавлен `urgent` (ранее backend отвергал его).

### Мелкие правки

- Фронтенд: тесты `TaskCard` обновлены под новый prop `members`.
- Calendar view: `TaskCard` и `TaskModal` обновлены под новые интерфейсы.
