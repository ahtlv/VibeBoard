# Changelog — 1 мая 2026

## Ambient Sound Mixer — полная переработка

### Реальные звуки вместо синтеза

Заменили генерацию звуков через Web Audio API на реальные аудиофайлы. Подключены: дождь, океан, лес, кафе, костёр, бинауральный шум 40 Гц.

### Наложение звуков

Раньше можно было включить только один звук. Теперь — любое количество одновременно. У каждого активного звука свой слайдер громкости.

### Перенос в сайдбар

Управление звуком переехало из Pomodoro-поповера (FAB) в левый сайдбар — постоянно доступно, не требует открытия панели таймера. Блок расположен над именем пользователя и кнопкой «Выйти», визуально отделён разделительной чертой.

### Кнопка Play/Pause с плавным переходом

Добавлена единая кнопка запуска микса. При нажатии звуки появляются с плавным нарастанием (~800 мс), при паузе — так же плавно затихают. Зелёный цвет кнопки — интуитивный сигнал «нажми, и начнётся что-то хорошее».

### Логика первого запуска

При открытии приложения ничего не выбрано. Первый клик на кнопку автоматически включает бинауральный шум 🧠 — рекомендуемый пресет для фокуса. Далее пользователь сам собирает сет.

### UX-детали

- Иконки без подписей — компактнее, меньше шума
- Бинауральный пресет: иконка 🧠 (вместо 🎵), стоит первым
- Кнопка Play полупрозрачно-зелёная — заметна, но не доминирует над интерфейсом
- Локализация: кнопка «▶ Звук» / «⏸ Пауза» на русском, «▶ Play» / «⏸ Pause» на английском
